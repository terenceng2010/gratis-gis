feat(web-map): SDF tinting for built-in point icons 
 
Built-in icons can now be tinted by the layer's fill color. The 
loader registers each icon as two MapLibre images: a plain raster 
(as-shipped colors) and an SDF variant (alpha-encoded distance 
field, tintable via icon-color). 
 
- Runtime chamfer distance transform in sdf.ts - no new deps, 
  ~100ms to process 25 icons at 96px at map init. 
- Schema: point.iconTint toggle, default on. 
- Canvas picks the plain or SDF variant per layer and emits 
  icon-color only when tinting is on. 
- Style editor: tint toggle + fill color picker appear under 
  the icon picker when an icon is selected. 
- Legend renders the icon in the tint color so the swatch matches 
  what's drawn on the map.
